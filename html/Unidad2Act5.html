HTML y Unicode

Para utilizar un caracter particular de un idioma utiliza el codigo UNICODE correspondiente precedido de "&#" y seguido de ; (punto y coma), por ejemplo, el codigo 21512, produce esto: 合

Ejemplos:

Código Caracter Descripcion
&#937; Ω Letra griega Omega
&#12354; あ Letra A hiragana (japones)
&#33865; 葉 Idiograma "Hoja" en chino tradicional

Pero tambien puede mostrar simbolos matematicos

  • 8734=∞ (Infinito)
  • 8721=∑ (Sumatoria)
  • 188=¼ (Fracción)
